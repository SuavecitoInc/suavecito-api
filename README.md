# Suavecito API v2

> NetSuite to Shopify API

## Setup

Environmental Variables:

```bash
# shopify
SHOPIFY_API_VERSION=2024-07
SECRET_KEY=
# retail
SHOPIFY_RETAIL_STORE=suavecito
SHOPIFY_RETAIL_API_KEY=
# wholesale
SHOPIFY_WHOLESALE_STORE=suavecito-wholesale
SHOPIFY_WHOLESALE_API_KEY=
```
